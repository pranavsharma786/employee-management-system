A Java-based Client-Server Employee Management System built using Socket Programming and Multithreading.

## Features

- Employee Record Management
- Client-Server Architecture
- Socket Communication
- Multithreaded Request Handling
- Attendance Management
- Performance Review System
- Employee of the Month Module

## Technologies

- Java
- Java Sockets
- Multithreading
- OOP
- JDBC / SQL
